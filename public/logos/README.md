# Network Logos

Place the official logo files for each ad network in this directory.

## Required Logos

The scrolling network section on the homepage expects the following logo files:

- `admob-logo.svg` or `admob-logo.png` - Google AdMob logo
- `applovin-logo.svg` or `applovin-logo.png` - AppLovin MAX logo  
- `ironsource-logo.svg` or `ironsource-logo.png` - ironSource logo
- `unity-logo.svg` or `unity-logo.png` - Unity Ads logo
- `chartboost-logo.svg` or `chartboost-logo.png` - Chartboost logo
- `vungle-logo.svg` or `vungle-logo.png` - Vungle logo

## Logo Specifications

- **Format**: SVG preferred (scales perfectly), PNG as fallback
- **Height**: Logos should be designed for 40px height (will scale responsively)
- **Background**: Transparent background
- **Color**: Full color (grayscale filter applied on hover off)
- **Aspect Ratio**: Maintain original aspect ratio

## Where to Get Logos

1. **Official Brand Resources**:
   - Google AdMob: https://developers.google.com/admob/branding

2. **Alternative Sources**:
   - Company press kit pages
   - Official marketing materials
   - Partnership/integration documentation

## Current Fallback

Currently, the website displays the network names as text if logo files are not found. Once you add the actual logo files, they will automatically appear in the scrolling marquee.

## Usage Example

If you place `admob-logo.svg` in this folder, the homepage will automatically display the Google AdMob logo instead of text.
